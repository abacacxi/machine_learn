# Relatório Final — Projeto de Machine Learning

## Predição de Desempenho Acadêmico

**Disciplina:** Machine Learning  
**Professor:** Durval Lins de Siqueira Neto  

---

## 1. Sumário Executivo

Este projeto tem como objetivo desenvolver um modelo de Machine Learning capaz
de prever o desempenho acadêmico de estudantes a partir de variáveis
educacionais. Ao longo do trabalho, foram realizadas etapas de análise
exploratória, pré-processamento dos dados, modelagem e otimização de
hiperparâmetros. O modelo final apresentou resultados satisfatórios nas métricas
de avaliação utilizadas.

---

## 2. Introdução

O desempenho acadêmico é um fator essencial para avaliar o aprendizado dos
estudantes. A utilização de técnicas de Machine Learning permite identificar
padrões nos dados educacionais e realizar previsões que podem auxiliar na tomada
de decisões pedagógicas. Este trabalho aplica modelos de regressão para prever a
nota final dos estudantes.

---

## 3. Análise Exploratória dos Dados

Nesta etapa, foi realizada a análise estatística descritiva do conjunto de dados,
bem como visualizações para compreender a distribuição das variáveis e suas
correlações. Foram analisados aspectos como valores médios, dispersão e relações
entre variáveis independentes e a variável alvo.

---

## 4. Pré-processamento dos Dados

O pré-processamento envolveu a limpeza dos dados, tratamento de valores
ausentes, seleção de atributos relevantes e divisão do conjunto de dados em
treino e teste. Essas etapas garantem que o modelo seja treinado de forma
consistente e confiável.

---

## 5. Modelagem

Foram testados modelos de regressão utilizando algoritmos de aprendizado
supervisionado. O desempenho dos modelos foi avaliado com métricas apropriadas
para regressão, permitindo a escolha daquele que apresentou melhores
resultados iniciais.

---

## 6. Otimização de Hiperparâmetros

Após a escolha do modelo base, foi aplicada a técnica de otimização de
hiperparâmetros utilizando Grid Search. Essa etapa permitiu ajustar os
parâmetros do modelo para melhorar seu desempenho e reduzir o erro de previsão.

---

## 7. Conclusões

O modelo final treinado foi capaz de prever o desempenho acadêmico com um nível
adequado de precisão. Os resultados demonstram o potencial do uso de Machine
Learning na área educacional. Como trabalhos futuros, sugere-se a inclusão de
mais variáveis e a avaliação de outros algoritmos.
